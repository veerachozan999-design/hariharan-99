# 🚗 Hill Racer

Hill Racer is a 2D hill racing game developed in Python using Pygame. The game features realistic hill terrain, vehicle physics, fuel management, AI opponent, vehicle customization, and multiple levels.

## ✨ Features

- 🚗 Smooth hill racing gameplay
- 🏁 10 unique levels
- ⛽ Fuel system
- 🪙 Coin collection (25, 50, 100, 500)
- 🤖 AI opponent
- 🚘 Vehicle customization
- 🔧 Upgrade system
  - Engine
  - Suspension
  - Tyres
- 🌄 Procedural terrain generation
- ☁️ Parallax background
- 💨 Particle effects
- 🔊 Procedural sound effects
- 📊 Speedometer
- 📍 Distance meter
- 🎯 Level unlock system
- ⚙️ Settings menu

## ▶️ How to Run

1. Install Python 3

2. Install Pygame

```bash
pip install pygame
```

3. Run the project

```bash
python hill_racer.py
```
